### Installation
`npm i` <br>

### ATA Server (Cloud)
`npm start` or `node server` <br>

### ATA Tracker (Local Machine - for debugging)
`npm run tracker` or `node tracker`

### ATA Remote (Local Machine - sends commands to the server)
`npm run remote` or `node remote`
